## 🧠 Overview

**JJK Cursed Tech** is a real-time cursed energy visualization system inspired by *Jujutsu Kaisen (呪術廻戦)*.

It merges:
- ✋ MediaPipe Hand Tracking
- 🎮 Three.js WebGL rendering
- ⚡ Real-time gesture recognition

to simulate interactive cursed techniques controlled entirely by hand movements.

Powered by **Google Gemini 3** and refined with **ChatGPT 5.4**.

## ✨ Core Features

A gesture-driven cursed energy simulation system built with GPU-based rendering.

---

### 🟣 Hollow Purple
A destructive singularity formed by merging attraction and repulsion forces.
- **Gesture:** Pinch (Thumb + Index)

---

### 🌌 Infinite Void (Domain Expansion)
A multi-layered cosmic perception field simulating infinite data flow.
- **Gesture:** Cross fingers (Index + Middle)

---

### 🔴 Cursed Technique Reversal: Red
A high-energy repulsive blast with intense force output.
- **Gesture:** Index finger pointing up

---

### ⛩ Malevolent Shrine (Domain Expansion)
A destructive cursed aura representing overwhelming domain pressure.
- **Gesture:** Open palm / prayer pose

---

## 🛠 Tech Stack

- **Three.js** — GPU-based 3D rendering & particle systems
- **MediaPipe Hands** — Real-time hand tracking
- **WebGL** — Hardware-accelerated graphics
- **JavaScript (ES Modules)**

## 🚀 Getting Started

### 📌 Requirements
- Modern browser (Chrome / Brave recommended)
- Webcam enabled
- Stable lighting for hand tracking

---

### 📦 Installation

```bash
git https://github.com/xjly1/JJK-cursed-tech.git
cd JJK
```

---

### ▶️ Run Locally

1. Open project in **VS Code**
2. Install **Live Server (Five Server) extension**
3. Right-click `index.html`
4. Click **Open with Live Server (Five Server)**

---

## ⚠️ Notes

- Runs entirely in the browser (no backend)
- Requires webcam permission
- Best performance on GPU-enabled browsers
